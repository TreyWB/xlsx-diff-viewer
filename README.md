# XLSX Diff Viewer

> Made with [Claude Code](https://claude.ai/code) â€“ using Opus 4.5

A lightweight web application that lets you drag-and-drop two Microsoft Excel workbooks and instantly see a color-coded side-by-side diff.

## Quick Start
1. Clone or download this repo
2. Start server (e.g. npx start)
3. Open http://localhost:3000
4. Drop two `.xlsx` files and watch the diff appear instantly

##Development
```bash
npm install
npm run dev      # start dev server
npm run build    # create production bundle
npm run preview  # test the built version
```
